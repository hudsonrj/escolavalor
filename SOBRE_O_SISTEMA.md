# EscolaValor - Encontre a escola ideal com transparência total

## 🎯 O que é

**EscolaValor** é uma plataforma gratuita que ajuda famílias a encontrar a melhor escola com base em **custo-benefício real**, não em marketing ou propaganda.

## ✨ O que traz de fantástico

### 1. **ICB - Índice de Custo-Benefício**
Responde a pergunta que ninguém responde: **"Quanto custa cada ponto de qualidade?"**

- 🏆 **R$ 3.097/ponto** - Excelente custo-benefício
- 💰 **R$ 15.695/ponto** - Mensalidade alta para a qualidade oferecida
- 📊 Compare escolas caras com públicas federais de graça

### 2. **Transparência Total**
- ✅ **Dados Oficiais INEP** - Quando disponível, mostramos a fonte
- ⚠️ **"Dados não disponíveis"** - Nunca mentimos
- 🔍 **Código aberto** - Veja como calculamos tudo

### 3. **Dados Reais, Não Inventados**
- ❌ **Removemos 100% dos dados sintéticos** que existiam
- ✅ **Apenas dados verificados** de fontes oficiais (INEP/MEC)
- 📈 **3 escolas com dados reais** (Pedro II, CEFET, CAp UFRJ) - expandindo!
- 🎓 **3.478 escolas cadastradas** (2.246 no RJ)

### 4. **Ranking Justo**
Compara escolas públicas e privadas de forma justa:
- Federais de excelência (Pedro II, CEFET) aparecem no topo
- Privadas caras só ficam bem se entregarem qualidade proporcional
- Escolas públicas de qualidade ganham destaque merecido

### 5. **Mobile First + PWA**
- 📱 Interface responsiva perfeita para celular
- 💾 **Instalável como app** (sem Play Store/App Store)
- 🚀 Rápido e leve
- 🔔 Futuro: notificações de novas escolas na sua região

## 🎁 O que é gratuito (tudo!)

- ✅ Acesso 100% gratuito
- ✅ Sem cadastro obrigatório
- ✅ Sem anúncios
- ✅ Sem limites de consulta
- ✅ Código aberto (GitHub)
- ✅ Dados públicos do INEP/MEC

## 💡 O que é útil

### Para Famílias:
- **"Vale a pena pagar R$ 4.000/mês nessa escola?"**
  - Compare com escolas públicas federais de graça
  - Veja o ICB real: pode ser que não valha

- **"Qual a melhor escola pública da região?"**
  - Ranking transparente por município
  - Dados reais de ENEM, olimpíadas, aprovações

- **"Meu filho precisa de qualidade, mas tenho orçamento limitado"**
  - Filtre por faixa de mensalidade
  - Ordene por ICB (melhor custo-benefício)
  - Descubra pérolas escondidas

### Para Escolas:
- Transparência aumenta confiança
- Escolas públicas de qualidade ganham reconhecimento
- Escolas privadas com bom custo-benefício se destacam

## 🚀 Tecnologia

- **Backend:** Node.js (Bun) + PostgreSQL + Hono
- **Frontend:** React + TypeScript + Tailwind CSS + shadcn/ui
- **PWA:** Service Worker + Manifest
- **Deploy:** Nginx reverse proxy
- **Dados:** INEP, MEC, Censo Escolar
